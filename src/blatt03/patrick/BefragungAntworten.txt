Antworten auf Fragen:

Aufgabe 3.2: Nennen Sie bitte zwei Schlüsselworte von Java,
die Sie in den letzten Übungsblättern verwendet haben,
die typisch für objektorientierte Sprachen sind:

-Schlüsselwort 1: new - Erstellen eines neuen Objekts

-Schlüsselwort 2: this - Zuweisung eines Wertes auf ein Objekt

Aufgabe 3.3: Wieso muss ein Java-Programm nur ein mal kompiliert
werden und kann dann auf verschiedenen Plattformen
(Unix, Windows, ...) eingesetzt werden?

-Beim Compilieren wird ein Maschinencode (Bytecode) erstellt, der
maschinenunabhängig ist und von jeder Plattform ausgeführt werden kann.


Aufgabe 3.4: Recherchieren Sie bitte, was unter Android
an Java anders ist:

-Java unter Android enthält nicht alle Standard-Bibliotheken
-Java unter Android verwendet eine andere VM zum Ausführen des Programms.


Aufgabe 3.9:
1. Wozu dient die paint()-Methode?

-Die Paint-Methode zeichnet Linien auf die Grafikoberfläche g

2. Wozu dient der Parameter g?

-Der Parameter g steht für das offene Fenster, wo die Zeichnungen
hinkommen.
